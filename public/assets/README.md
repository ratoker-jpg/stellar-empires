# Runtime assets

Все финальные игровые ассеты размещаются здесь и копируются Vite в production-сборку без изменения имён.

## Структура

```text
assets/
├── branding/
├── backgrounds/
├── factions/
│   ├── aegis/
│   ├── synod/
│   └── veyra/
├── planets/
├── shared/
│   ├── effects/
│   ├── icons/
│   └── ui/
└── audio/
```

До утверждения P0 используются программные заглушки. Чужие ассеты и материалы с неясными правами добавлять запрещено.
