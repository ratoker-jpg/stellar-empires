# PR #36 — Colonization technology and mission

The Aegis research tree now includes Colonization Protocols. Level 1 unlocks colonization expeditions, and each level raises the empire colony limit by one above the home planet.

A colonization mission requires an Aegis colony ship, an available non-gas galaxy planet and free colony capacity. Flight distance is calculated directly to the unoccupied galaxy planet. Arrival revalidates ownership and capacity so competing expeditions cannot create duplicate colonies.

Successful expeditions update the galaxy owner, create a complete planet state with starter command, extraction and power infrastructure, unload expedition cargo and consume one colony ship. Escort ships are stationed at the new colony. If the target becomes unavailable before arrival, the expedition returns safely.

Active colonization missions are accepted by save migration and validation. The mission screen exposes free worlds and the current colony limit.
