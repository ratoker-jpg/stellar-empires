# Game Design

**Статус:** Draft  
**Последнее обновление:** 2026-07-18

## 1. Игровой цикл

Базовый цикл:

1. Получить ресурсы.
2. Выбрать развитие планеты.
3. Построить здание или начать исследование.
4. Создать корабли или оборону.
5. Получить информацию о галактике.
6. Выбрать цель.
7. Отправить флот.
8. Получить отчёт и изменить план.

Цикл должен работать и для игрока, и для ботов.

## 2. Масштаб мира

Предварительная модель:

- галактика состоит из систем;
- система содержит несколько позиций для планет;
- одна империя может владеть несколькими планетами;
- расстояние влияет на время полёта и расход топлива;
- часть позиций свободна, часть занята ботами, часть содержит нейтральные объекты.

Настройки новой партии:

- размер галактики;
- число бот-империй;
- скорость мира;
- сложность ботов;
- плотность свободных планет;
- агрессивность мира;
- начальный seed генерации.

## 3. Ресурсы

Начальный набор:

| Ресурс | Назначение |
|---|---|
| Металл | Основной материал зданий и кораблей |
| Кристалл | Электроника, исследования, продвинутые корабли |
| Газ | Топливо, двигатели, часть технологий |
| Энергия | Ограничивает активность производственных зданий |

Правила:

- добыча считается непрерывно;
- производство зависит от уровней зданий;
- нехватка энергии снижает эффективность;
- склады ограничивают максимальный объём;
- ресурсы хранятся отдельно на каждой планете;
- транспортировка требует грузовых кораблей.

## 4. Планета

Планета содержит:

- размер и число строительных полей;
- ресурсные здания;
- промышленные здания;
- исследовательские здания;
- верфь;
- оборону;
- локальные склады;
- очередь строительства;
- очередь производства;
- локальные модификаторы.

Планеты могут специализироваться:

- добывающая;
- исследовательская;
- промышленная;
- военная;
- логистическая;
- укреплённая.

## 5. Строительство

Каждое здание задаётся данными:

- идентификатор;
- название;
- категория;
- максимальный уровень;
- стоимость по уровням;
- время строительства;
- требования;
- энергопотребление или производство;
- игровой эффект.

Первая версия использует одну очередь строительства на планету.

Отмена строительства возвращает только часть ресурсов. Точный коэффициент задаётся балансом.

## 6. Исследования

Исследования открывают:

- новые здания;
- типы кораблей;
- улучшения добычи;
- скорость полёта;
- дальность разведки;
- боевые характеристики;
- дополнительные колонии.

Исследования принадлежат империи, но запускаются на конкретной планете.

Первая версия использует одну общую очередь исследований на империю.

## 7. Корабли

Минимальные классы первой версии:

| Класс | Роль |
|---|---|
| Разведчик | Получение информации |
| Малый транспорт | Перевозка ресурсов |
| Большой транспорт | Массовая логистика |
| Лёгкий истребитель | Дешёвая ударная единица |
| Тяжёлый истребитель | Бой с более защищёнными целями |
| Фрегат | Средний универсальный корабль |
| Крейсер | Основная тяжёлая ударная единица |
| Колонизатор | Основание новой планеты |
| Переработчик | Сбор обломков |

Каждый тип имеет:

- корпус;
- щит;
- атаку;
- скорость;
- грузоподъёмность;
- расход топлива;
- стоимость;
- время производства;
- приоритеты целей.

## 8. Флоты и миссии

Флот — группа кораблей, выполняющая одну миссию.

Миссии первой версии:

- разведка;
- атака;
- транспорт;
- передислокация;
- колонизация;
- переработка обломков.

Жизненный цикл флота:

1. Проверка доступности кораблей и топлива.
2. Расчёт времени и стоимости.
3. Создание события прибытия.
4. Выполнение миссии.
5. Создание отчёта.
6. Возврат или размещение на целевой планете.

## 9. Разведка

Результат зависит от:

- уровня разведывательной технологии;
- числа разведчиков;
- контрразведки цели;
- случайного, но воспроизводимого коэффициента.

Уровни информации:

1. Базовые сведения о планете.
2. Ресурсы.
3. Здания.
4. Исследования.
5. Флот и оборона.
6. Полная оценка боевой силы.

Неудачная разведка может быть обнаружена.

## 10. Бой

Бой должен быть детерминированным при одинаковом seed и входных данных.

Этапы:

1. Формирование сторон.
2. Применение технологий и модификаторов.
3. Выбор целей.
4. Несколько раундов обмена уроном.
5. Определение победителя или ничьей.
6. Потери.
7. Образование поля обломков.
8. Захват допустимой части ресурсов.
9. Формирование отчёта.

На первом этапе не нужны сложные построения и ручное управление боем. Игрок управляет подготовкой и составом флота, а не каждым выстрелом.

## 11. Оборона

Оборонительные сооружения:

- остаются на планете;
- участвуют в бою вместе с флотом;
- не требуют топлива;
- могут частично восстанавливаться после боя;
- имеют собственные приоритеты целей.

## 12. Колонизация

Для новой колонии нужны:

- свободная позиция;
- технология колонизации нужного уровня;
- колонизатор;
- допустимый лимит планет;
- стартовый набор ресурсов.

Колония начинает с минимальной инфраструктуры и может получать помощь от других планет.

## 13. Отчёты и журнал

Игрок получает отчёты о:

- завершении строительства;
- завершении исследования;
- производстве кораблей;
- разведке;
- атаке;
- входящем флоте;
- колонизации;
- действиях ботов, доступных по разведданным.

Каждый отчёт должен хранить фактические входные данные и результат, чтобы ошибки можно было воспроизвести.

## 14. Условия победы

Первая версия может поддерживать несколько режимов:

- господство по числу контролируемых планет;
- уничтожение всех крупных соперников;
- достижение технологической цели;
- набор заданного количества очков;
- открытая бесконечная партия без победы.

Для первого вертикального среза условия победы не обязательны.

## 15. Принцип расширения

Новые механики добавляются только после стабильной работы базового цикла. Каждая механика должна иметь:

- правила;
- структуру данных;
- команды;
- события;
- отчёты;
- тесты;
- поведение ботов.
