## Rules for project use

- Prefer current project design and current `main` over external mechanics.
- Prefer newer official release notes over older help articles when researching history.
- Never import the external economy or combat values as final balance without an original decision and simulation tests.
- Never commit captured account HTML, screenshots, session data or downloaded third-party static assets.
- Translate useful patterns into original Stellar Empires systems: command doctrine, flagships, coalitions, strategic stars, anchor arrays and the final transit nexus.
