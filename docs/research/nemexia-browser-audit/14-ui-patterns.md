# 14 — UI patterns

Observed patterns (`CONFIRMED_UI`): persistent global header, resource/status strip, coordinate-aware planet context, primary navigation, zone grouping, tabbed shipyard categories, queue/time readout, and explicit requirement lock copy.

Adapt patterns, not visual expression. Stellar Empires must use its own typography, layouts, frames, iconography, terminology, art direction, and interaction code. Screens are evidence only and must not be used as a design copy source.
