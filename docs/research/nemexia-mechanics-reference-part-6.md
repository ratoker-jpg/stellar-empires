# 27. Буквальные ограничения исследования

Документ закрывает механику всех 96 пунктов меню, но не подменяет недоступные данные догадками.

1. **Transport, Blueprints и Loyalty program** — текущие тела страниц официальный сервер не отдал. Содержание восстановлено по другим официальным статьям, release notes и старому официальному зеркалу. В матрице они отмечены B.
2. **Формулы Battle Points** — две точные формулы встроены в изображения без текстового слоя. Подтверждённая логика сохранена, выражения не выдуманы.
3. **Image-only таблицы сервисов** — часть Premium, Platinum, Subscriptions и старых сервисов представлена изображениями. Все извлекаемые текстовые правила включены; нечитабельные числа не реконструировались.
4. **Визуальные материалы** — этот справочник описывает механику. Арты, рамки и декоративные изображения не встроены.
5. **Расхождения с живой игрой** — при конфликте более новый release note принят как ориентир, старое значение сохранено в таблице конфликтов.
6. **Динамические значения** — турниры, сервисы, предложения и капсулы могут меняться без полной синхронизации справочника.

# 29. Архитектурный вывод

```text
планетарная экономика
→ инфраструктурные требования
→ глобальная наука
→ производство и Population
→ флот и полёты
→ разведка, PvP и PvE
→ Scrap, ремонт и переработка
→ Admiral и Commander Ships
→ альянсовая прогрессия
→ Sun Battles
→ кристаллы и Обелиски
→ Supreme Starway Gate
→ завершение раунда
```

Главная особенность — взаимная связанность прогрессий. Для Stellar Empires полезно сохранять структурную глубину, но не копировать названия, визуальные решения и буквальные балансовые числа без отдельного пересчёта.
